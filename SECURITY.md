# Security Policy

If you discover a security vulnerability, please do not create a public issue.

Instead, use the private vulnerability reporting feature provided by GitHub:

- Go to the [Security and Quality](https://github.com/Lifailon/lazydebugger/security) tab.
- Click the `Report a Vulnerability` button to open a private security advisory draft.

We will review the vulnerability and keep you updated on the fix. Thank you for helping keep this project secure!
